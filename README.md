# Teste Prático

Consiste em um **simples projeto webapp** para teste de aprendizado usando **HTML, CSS e JAVASCRIPT**, sem a utilização de nenhuma outra linguagem como backend(PYTHON, PHP, C# etc...).

---

## Principal funcionalidade

Sua principal função é buscar em uma **API pública em formato json**** e com as informações obtidas renderizar em formato de tabela as informações tornando assim mais organizada e apresentável para o usuário.

---

## IDE e Linguagens usadas

- **Visual Studio Code**
    - **HTML**
    - **CSS**
    - **JAVASCRIPT**

---

## Como Executar:

1. Clone o repositório ou baixe manualmente cada arquivo.
**IMPORTANTE:** Para o webapp funcionar, todos os arquivos devem estar no mesmo lugar, recomendo deixar-los agrupados em uma pasta.

2. Para conseguir abrir o webapp basta abrir o arquivo `.html` no seu navegador de preferência que o mesmo irá funcionar, lembrando que caso os arquivos não estejam juntos em um único lugar, o mesmo pode apresentar erros ou não funcionar.

---

## OBSERVAÇÕES

- O projeto utiliza uma **API pública** para buscar dados
- Não há necessidade de um servidor ou a necessidade de alguma dependência.
